性格

关于性格，我并不想用冷冰冰的 MBTI 标签来定义自己。

我是一个完美主义者，甚至带着一点强迫症，无法容忍指尖流出的代码或作品留下瑕疵。在团队中，我自认为领导力尚可，习惯将事务打理得井井有条。

现实中的我不爱说话，抵触社交。在陌生人面前，我会习惯性地紧张。这种性格让我有时觉得自己像《冰菓》里的折木奉太郎——冷静、理智、甚至有些沉默。但这并非刻意模仿，我只是更享受安静，享受与书籍、代码、游戏为伍的纯粹。

学业与 OI

在人才流失严重的小县城，我的学习之路还算顺利。从小学时的懵懂，到初中时顶着压力考入直升班，我的成绩逐渐稳定。

我并非天才，目前的目标也很现实：考上一所 211 大学。我相信只要步履不停，梦想终会近在咫尺。

关于 OI（信息学竞赛），我的接触来得有些太晚。去年二月才起步的我，面对复杂的算法和思维逻辑，确实感受到了不小的阻力。但我并不打算放弃，每天的练习是我对这份“起步晚”的补救。坦白说，学习 OI 的初衷带有功利色彩——为了升学，为了在这个赛道上换取一个挑战自我的机会。

技能树

相比文化课，这些我自学掌握的计算机知识，才是我至今做过最有意义的事。从小学二年级开始，我一点点搭建起了自己的技能版图。

在编程语言的领域里：

  • 我能熟练运用 C++ 解决逻辑问题，用 HTML 与 Markdown 构建内容；
  • 同时也对 Python、CSS 、Rust 和 Go 保持着探索的习惯。

在工具与创作方面：

从 Windows 全家桶到各种 Linux 发行版（尤其是 Arch Linux），我享受这种掌控系统的快感。我也折腾硬件、当过“垃圾佬”，这些经历让我对这个世界有了更底层、更透彻的理解。

娱乐与旅行

我的精神世界由动漫、电影、游戏组成。

我钟爱《冰菓》、《春物》这种青春与感伤交织的作品。但电影才是我的心头好——300 多部的阅片量，带我穿越了无数段人生。无论是《少年派的奇幻漂流》，还是《肖申克的救赎》，都曾深刻地触动过我。

至于游戏，我追求的从未不是竞技，而是沉浸感。我喜欢在《死亡搁浅》中感受那份宏大的孤独，在《孤岛惊魂》里释放压力，或是在 P 社的游戏中消磨一整天的时光。

我走过的地方不多，上海的夜景曾让我震撼，外滩的霓虹与弄堂的古老交错，那是现代与历史的碰撞。我遗憾于还未曾登顶泰山，也遗憾于为了新电脑而推迟了去东京圣地巡礼的计划。但我相信，旅行不仅是风景，更是为了在不同的文化中开阔眼界。

遗憾与展望

遗憾当然有。
